# mterm

An electron terminal written with React and TypeScript.

![image](https://github.com/mterm-io/mterm/assets/7341502/3009f033-2e5f-4ba7-a965-46a575200fbd)



## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
